  TaxPro CA Platform - QUICK START
════════════════════════════════════════════════════════════════════════════════

Your enterprise CA firm platform is complete and ready to run.

────────────────────────────────────────────────────────────────────────────────
 1. INSTALL DEPENDENCIES (1 minute)
────────────────────────────────────────────────────────────────────────────────

Open PowerShell in the project directory and run:

  cd "E:\website\CA firm\ca-firm-platform"
  ..\setup.ps1

OR manually:

  npm install --legacy-peer-deps

This installs Next.js 14, React, Prisma, Supabase, authentication, UI components,
and all other dependencies.

────────────────────────────────────────────────────────────────────────────────
 2. SET UP DATABASE (2 minutes)
────────────────────────────────────────────────────────────────────────────────

Easiest: Use Supabase (free account at supabase.com)
  • Create project
  • Copy PostgreSQL connection string
  • Paste into .env.local as DATABASE_URL

Other option: Use local PostgreSQL
  • Create database: createdb ca_firm_db
  • Connection string: postgresql://postgres:password@localhost:5432/ca_firm_db

────────────────────────────────────────────────────────────────────────────────
 3. CONFIGURE ENVIRONMENT (2 minutes)
────────────────────────────────────────────────────────────────────────────────

Create .env.local file with:

  DATABASE_URL="your-postgres-connection-string"
  AUTH_SECRET="your-random-32-char-string"
  AUTH_URL="http://localhost:3000"
  NEXT_PUBLIC_SUPABASE_URL="https://your-project.supabase.co"
  NEXT_PUBLIC_SUPABASE_ANON_KEY="your-supabase-key"
  SUPABASE_SERVICE_ROLE_KEY="your-service-role-key"

See .env.local (already created) for all available options.

────────────────────────────────────────────────────────────────────────────────
 4. INITIALIZE DATABASE (1 minute)
────────────────────────────────────────────────────────────────────────────────

  npm run db:generate
  npm run db:push
  npm run db:seed

This creates tables and seeds demo data:
  • Admin account: admin@taxpro.in / Admin@123456
  • CA Staff: ca.rajesh@taxpro.in / Staff@123456
  • Client: client@techventures.in / Client@123456

────────────────────────────────────────────────────────────────────────────────
 5. START DEVELOPMENT SERVER (done)
────────────────────────────────────────────────────────────────────────────────

  npm run dev

Open: http://localhost:3000

You'll see:
  • Public website at /
  • Login page at /auth/login
  • Client portal at /dashboard (after login)
  • Admin dashboard (with admin account)

All source code is in src/
  • src/app/ — Next.js pages and layouts

Configuration files:
  • package.json — Dependencies
  • tsconfig.json — TypeScript config
  • tailwind.config.ts — Styling

  • prisma/schema.prisma — Database schema
