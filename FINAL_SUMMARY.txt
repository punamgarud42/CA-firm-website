HOW TO RUN - STEP BY STEP
════════════════════════════════════════════════════════════════════════════════

STEP 1: OPEN TERMINAL
─────────────────────
Location: E:\website\CA firm\ca-firm-platform

Press Windows Key + R, type "powershell", navigate to project:
  cd "E:\website\CA firm\ca-firm-platform"


STEP 2: INSTALL DEPENDENCIES (3 minutes)
──────────────────────────────────────
  npm install --legacy-peer-deps

This downloads Next.js, React, Prisma, Supabase, and all other dependencies.


STEP 3: SET UP DATABASE (Choose ONE option)
─────────────────────────────────────────

OPTION A: SUPABASE (Easiest - Recommended)
  1. Go to https://supabase.com
  2. Create free account
  3. Create new project
  4. Go to Settings → Database
  5. Copy PostgreSQL connection string
  6. Create file ".env.local" in project root with:

    DATABASE_URL="postgresql://postgres:PASSWORD@PROJECT.supabase.co:5432/postgres"
    AUTH_SECRET="your-random-32-character-string"
    AUTH_URL="http://localhost:3000"
    NEXT_PUBLIC_SUPABASE_URL="https://PROJECT.supabase.co"
    NEXT_PUBLIC_SUPABASE_ANON_KEY="your-anon-key"
    SUPABASE_SERVICE_ROLE_KEY="your-service-role-key"

  (Replace PROJECT, PASSWORD, and keys with your actual values)


OPTION B: LOCAL POSTGRESQL
  1. Download from https://www.postgresql.org/download/windows/
  2. Install and remember password
  3. Open Command Prompt and run:
     psql -U postgres
     CREATE DATABASE ca_firm_db;
     \q

  4. Create ".env.local" in project root:
     DATABASE_URL="postgresql://postgres:YOUR_PASSWORD@localhost:5432/ca_firm_db"
     AUTH_SECRET="your-random-32-character-string"
     AUTH_URL="http://localhost:3000"


STEP 4: INITIALIZE DATABASE (1 minute)
───────────────────────────────────
Run in PowerShell:
  npm run db:generate
  npm run db:push
  npm run db:seed

This creates tables and adds demo accounts:
  - admin@taxpro.in / Admin@123456
  - ca.rajesh@taxpro.in / Staff@123456
  - client@techventures.in / Client@123456


STEP 5: START DEVELOPMENT SERVER
────────────────────────────────
  npm run dev

Wait for message:
  "✓ compiled client and server successfully"

Then open: http://localhost:3000


DONE! You're running the platform! 🎉

TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

Q: "Cannot find module 'next'"
A: Run: npm install --legacy-peer-deps

Q: "Database connection failed"
A: Check .env.local DATABASE_URL. For Supabase, use PostgreSQL connection string.

Q: "Port 3000 already in use"
A: Run: npm run dev -- -p 3001

Q: "Prisma type errors"
A: Run: npm run db:generate

Q: ".env.local file not found"
A: Create it manually with the values from .env.example

Q: "Compilation errors"
A: Run: npm install --legacy-peer-deps && npm run db:generate

Q: "OAuth/Google Sign-in not working"
A: Optional for now. Use email/password login. See RUN_GUIDE.md for OAuth setup.
